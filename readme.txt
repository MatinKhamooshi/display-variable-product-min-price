Displays the minimum price of variable products as the default price on all front pages. Ensures the price is greater than 0.

== Description ==

This plugin modifies the display of variable product prices in WooCommerce. It shows the minimum price of all available variations as the default price on all front pages, while ensuring that only prices greater than 0 are considered.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/display-variable-product-min-price` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. The plugin will automatically modify the price display for variable products

== Frequently Asked Questions ==

= Does this plugin work with all WooCommerce themes? =

Yes, this plugin should work with any WooCommerce compatible theme as it uses standard WooCommerce hooks and filters.
